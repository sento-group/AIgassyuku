# 📊 ライン管理システム - クイックスタートガイド

## 🚀 今すぐ始める

### ステップ1: システムを開く

ブラウザで `index.html` を開いてください。

または、お気に入りの機能に直接アクセス：
- メインダッシュボード: `pages/dashboard.html`
- 稼働日提出: `pages/availability-submit.html`
- KPI管理: `pages/kpi-dashboard.html`
- 実績管理: `pages/performance.html`

---

## 📁 5分でわかる各機能

### 1️⃣ メインダッシュボード

**使い方:**
1. サイドバーの「ダッシュボード」をクリック
2. KPIカードで現状を確認
3. クイックアクションボタンで自動配置やデータ出力

**便利な機能:**
- 検索バーでスタッフや案件を瞬時に検索
- スキルフィルターで条件に合うスタッフを絞り込み
- グラフで視覚的に状況を把握

---

### 2️⃣ 稼働日提出

**使い方:**
1. カレンダーで稼働可能な日をクリック
2. 青色にハイライトされた日が選択済み
3. 「✅ 提出する」ボタンで確定

**ポイント:**
- 複数日を一度に選択可能
- 提出期限は毎月15日（アラート表示あり）
- 過去の提出データも閲覧可能

---

### 3️⃣ KPI管理

**使い方:**
1. 「KPI概要」タブで今月の目標と実績を確認
2. 「売上予測」タブで来月の見込みをチェック
3. 「事業計画」タブで年間目標を設定

**売上予測の仕組み:**
```
予測売上 = 平均単価 × 延べ稼働日数 × マッチング率
```

---

### 4️⃣ 実績管理

**使い方:**
1. スタッフ・案件・実施日を選択
2. MNP、光、カード、端末の獲得数を入力
3. 評価（⭐1〜5）を選択
4. 「📝 実績を登録」で完了

**出力機能:**
- 📄 PDF出力: スキルシートをPDFで保存
- 📊 Excel出力: データをExcel形式でダウンロード
- 📷 画像出力: PNG形式で保存
- 📝 テキスト出力: クリップボードにコピー

## 💡 よくある質問

### Q1: データはどこに保存されますか？
A: ブラウザのローカルストレージに保存されます。ブラウザを閉じてもデータは残ります。

### Q2: スマホから使えますか？
A: はい。レスポンシブ対応しており、スマホ・タブレットからも利用可能です。

### Q3: データをバックアップするには？
A: ブラウザのデベロッパーツール（F12）→ Application → Local Storage で確認・エクスポートできます。

### Q4: 複数人で共有できますか？
A: 現在はローカルストレージのため、各ブラウザで独立しています。今後、データベース連携を予定しています。

## 🎨 カスタマイズ

### カラーテーマを変更する

`assets/css/dashboard-v2.css` の `:root` セクションを編集：

```css
:root {
    --navy-blue: #1A3A52;         /* お好みの色に変更 */
    --turquoise-blue: #00A0B0;    /* お好みの色に変更 */
}
```

### データを追加する

`assets/data/data-100.js` を編集して、スタッフや案件を追加できます。

## 🆘 トラブルシューティング

| 問題 | 解決方法 |
|------|---------|
| データが表示されない | ブラウザのキャッシュをクリア（Ctrl + F5） |
| スタイルが崩れる | `assets/` フォルダのパスが正しいか確認 |
| モバイルでメニューが出ない | ハンバーガーアイコン（☰）をタップ |
