# システム使用ガイド

## クイックスタート

### 1. 支社長データベース

```bash
cd システム/支社長データベース
python3 cli.py add 沖縄県 "山田太郎" "経歴情報"
python3 cli.py list
python3 cli.py get 沖縄県
```

### 2. 提案資料作成（キャプテンズグループ様向け）

```bash
cd システム/提案資料作成システム
python3 create_proposal.py
```

生成されるファイル：
- `資料/01_提案資料/【キャプテンズグループ】提案資料_YYYYMMDD.json`
- `資料/01_提案資料/【キャプテンズグループ】提案資料_YYYYMMDD.md`

### 3. 月次レポート生成

```python
from report_generator import SNSReportGenerator

generator = SNSReportGenerator()

# クライアント登録
client_id = generator.add_client(
    client_name="キャプテンズグループ",
    company_url="https://www.captains-g.co.jp/",
    industry="ホテル・リゾート"
)

# SNSアカウント登録
account_id = generator.add_sns_account(
    client_id=client_id,
    platform="Instagram",
    account_name="captain_yoshiji",
    account_url="https://www.instagram.com/captain_yoshiji/"
)

# データ登録
generator.add_sns_data(
    account_id=account_id,
    report_date="2025-09",
    followers=5000,
    engagement_rate=3.5,
    reach=15000
)

# レポート生成
report = generator.generate_monthly_report("キャプテンズグループ", 2025, 9)
generator.save_report(report)
```
