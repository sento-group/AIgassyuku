**■ アレンジレシピ**

| 用途 | プロンプト追加文 |
|------|------------------|
| **業界特化** | 「○○業界の専門用語を使って」 |
| **緊急対応** | 「至急対応が必要な課題として強調して」 |
| **予算重視** | 「コストパフォーマンスを最重視して」 |
| **短文版** | 「A4用紙1枚、1000文字以内で」 |

**■ 失敗しないための3つの秘訣**

1. **顧客情報は具体的に**：「製造業」ではなく「自動車部品製造業」
2. **数値は必ず入れる**：「効率化」ではなく「作業時間50%短縮」  
3. **最後は人間が仕上げる**：AIは80点のたたき台、あなたが100点に！

**■ よくある質問＆解決法**

| 質問 | 解決法 |
|------|--------|
| 思った内容にならない | プロンプトに具体例を追加 |
| 専門用語が足りない | 「○○業界の用語で」を追記 |
| 文章が硬い | 「親しみやすいトーンで」を追記 |
| 短くしたい | 「○○文字以内で」を明記 |
