## 00:10:02 Step1: GPTプロジェクトの作成

まず、ChatGPTでプロジェクトを新規作成しました。

### 使用したプロンプト（多角的分析用）

```
ChatGPTのパーソナライズ設定で私の会社についての情報を設定するのに最適な調査検索を実行してその内容を構造的に出力してください。
どのような仕事をしているかなど具体的にわかればなお良いです。
```

```
最近のプレスリリースとかニュースとか発信があれば教えて
特にTwitterとかでもOKです
```

```
諸々調査情報をベースにしてSWOT分析や3C分析、ビジネスモデル・キャンバスなんかを使って多角的にsento.groupについて分析してください。
んで、分析したあと、私との対話を通じて、私の理解を深める対話をしてください。
追加で調べる必要があれば追加で調べて

調査対象は
和島祐生
会社はsento.group
URLはhttps://sento.group/
Xはhttps://x.com/ElephantWajima4
```

**実例デモ**: 「先頭グループのデータコンサルの競合調査」で実演し、AIが自動的に多角的分析を出力することを確認しました。

## 00:15:55 Step2: Web検索機能の活用

ChatGPTの左下プラスボタンからWeb検索を選択し、最新情報を取得しました。

### 最新情報取得プロンプト

```
先頭グループという会社と打ち合わせをしに行く前に、最近のニュースや話題になりそうなことを教えて
```

**追加ツール紹介**: 
- ディープリサーチ機能（詳細情報取得、時間はかかる）
- Grok（TwitterのX情報検索に強い）

## 00:19:08 便利ツール: AIブラウザ「リア」紹介

画面操作自動化、検索履歴・関連リンク候補表示で効率化を実現。  
**デモ**: 「スプレッドシート」入力で関連ファイル候補が自動表示

## 00:23:06 Step3: 競合分析とSWOT分析の実践

競合分析テンプレートを使って、SWOT分析やファイブフォース分析を実行しました。

### 競合分析プロンプト

```
[対象企業名]について、以下の項目で競合分析レポートを作成してください：

1. 企業概要（強み、市場状況）
2. 競合他社の特定と比較
3. SWOT分析
4. ファイブフォース分析
5. セールストークに活用できる要素

構造的に出力し、実際の商談で使える形でまとめてください。
```

**成果**: AIが企業の強み、市場状況、脅威を含む包括的なレポートを作成し、これをセールストークのたたき台として活用できることを確認しました。

## 00:29:29 Step4: パーソナライズ設定の活用

ChatGPTの左下アカウント名 → 設定 → カスタム指示でパーソナライズ設定を行いました。

### 自分情報調査プロンプト

```
和島祐生について詳細に調査し、仕事内容、専門性、会社情報を構造的に出力してください。
これをChatGPTのパーソナライズ設定に反映させるため、適切な形式で整理してください。

調査対象:
- 名前: 和島祐生
- 会社: sento.group
- URL: https://sento.group/
- X: https://x.com/ElephantWajima4
```

**ポイント**: 自分の名前と立場を設定することで、AIが自分の立場で提案するトークスクリプトを作成可能になります。

## 00:47:09 Step5: セールストークの作成_イエスセット手法

SWOT分析結果を活用してセールストークを作成しました。

### セールストーク作成プロンプト

```
じゃぁこれベースに和島が提案するようなsento.groupの提案の概要を考えて
提案はあくまで自然な形で導入部分をイエスセットとDESC法を用いて相手の状況に理解を示しながら同意を得られる流れ導入としてください

これをそのままパーソナライズ設定の自分のパーソナリティとして設定したいので、それに適した文体に変換して
```

**イエスセット手法**: 5回のYesを積み重ねることで同意を得やすくする手法  
相手の状況を描写して同意を引き出し、自然な流れで提案に導く

## 00:59:43 Step6: メモリ管理設定

GPTのメモリ機能設定を行いました：
- チャット履歴を参照する設定をオン
- 記録モードをオンにして過去のやり取りを保存
- これにより一貫性のある対話が可能に

## 📝 実際に試してみるには

3. **競合分析実行** → SWOT・ファイブフォース分析

5. **セールストーク作成** → イエスセット手法で提案導入部分作成
6. **メモリ設定** → 継続的な学習と改善
