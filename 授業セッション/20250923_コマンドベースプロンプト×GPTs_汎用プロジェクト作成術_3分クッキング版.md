## 🎯 セッション概要

**テーマ**: ChatGPTのGPTsに設定するコマンドベースプロンプトで、様々なプロジェクトを段階的にガイドしてくれる汎用的なGPTsを作成

**習得スキル**: コマンドベースのプロンプト設計術、汎用的なGPTs作成、プロジェクト進行の自動化

## 💡 コマンドベース設計の基本原則

### 覚えやすいコマンド体系
```
/start   - プロジェクト開始
/plan    - 計画立案
/next    - 次のアクション
/check   - 進捗確認
/review  - 成果レビュー
/help    - サポート要請
```

### 汎用性を保つコツ
- プロジェクト種別に依存しない質問設計
- 段階的な情報収集で具体化
- ユーザーの状況に応じた柔軟な回答

## 📝 補足情報

### プロンプト設計テンプレート例
```
あなたは汎用的なプロジェクト進行支援アシスタントです。
コマンドベースで段階的にガイドを提供します。

基本コマンド:
- /start: プロジェクト概要を聞き、全体像を整理
- /next: 現在の状況から最適な次のステップを提案
- /check: 進捗状況を確認し、調整が必要な点を指摘
```

### よくある活用シーン
- **新規事業企画**: アイデア出しから事業計画まで
- **システム開発**: 要件定義から運用まで
- **マーケティング施策**: 戦略立案から効果測定まで
- **研修・学習プログラム**: カリキュラム設計から評価まで
