### 2. 4つの機能解説（20分）

### 機能❶ 全ソース一括同期 - Googleドライブ連携（5分）

**解決する課題:**

* ソースが10個、20個と増えると、一つずつ同期ボタンを押すのは苦行
* 同期漏れで古い情報のまま回答されてしまう

**使い方:**

```
【全ソース一括同期の手順】
1. NotebookLMを開く
2. 拡張機能で追加された「一括同期ボタン」をクリック
3. 全ソースの更新状況を確認し、一括で再同期実行

【ここが凄い】
- 数十個のファイルも一撃で最新化
- 同期漏れのリスクがゼロ
- リサーチ業務の「単純作業」が消滅

```

### 機能❸ ウェブクローラー（5分）

**解決する課題:**

* Webサイトの情報を1ページずつコピペ・PDF化は手間
* サイト全体の知識を一括でNotebookLMに入れたい

**使い方:**

```
【ウェブクローラーの手順】
1. 対象サイトのルートURLを入力
2. クローラーがリンクを辿って自動巡回
3. Markdown形式で1ファイルに結合・出力
4. NotebookLMにソースとして追加

【ここが凄い】
- Webサイト全体の情報を自動収集
- NotebookLMに読み込ませやすい形式で出力
- サイトを一撃で知識ベース化

```

### 機能❹ 同期スケジューラー（5分）

**解決する課題:**

* Googleドライブの資料を更新しても、NotebookLM側で再同期ボタンを押さないと反映されない
* 常に最新情報で対話したい

**使い方:**

```
【同期スケジューラーの設定】
1. 拡張機能の設定画面を開く
2. 同期間隔を選択：「1時間ごと」「毎日」「毎週」
3. バックグラウンドで自動同期開始

【ここが凄い】
- PCを開いておけば、勝手に裏で最新情報を取得
- 常に最新のデータベースでAIと対話可能
- 「情報が古い！」のストレスが完全消滅

```

## 💡 実践のコツ

### 🌈 拡張機能活用の原則

1. **まずは同期スケジューラーを設定**
   * 一度設定すれば放置でOK
   * 毎日使うなら「毎日」設定がおすすめ
2. **Studioパネルは普段は折りたたむ**
   * チャット中心の作業時は閉じておく
   * 音声生成時だけ開く
3. **クローラーは計画的に使う**
   * 大規模サイトは時間がかかる
   * 必要な部分だけに絞る
