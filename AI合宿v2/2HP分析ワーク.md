# 自社HP分析＆新HP作成ワーク

実習2（10:30-12:00）で使用する、自社HPを分析して新しいHPを作成するワークの手順書です。

**ミニマムゴール**：自社HPの分析結果と、新HPのプロトタイプHTMLが完成している

> **💡 昼休憩前の中間チェック**：ここまでで「AIサイクルを1回回した」状態になっていればOK

---

## 目的

- 自社HPを客観的に分析する
- 改善案を言語化する
- 新HPのプロトタイプを実際に作成する
- **AIサイクル（Input→加工→Output→Action）を実践で体感する**

---

## AIサイクルの流れ

```
【Input】自社HPのURL
    ↓
【加工】Cursorで分析・改善案作成
    ↓
【Output】新HPのプロトタイプHTML
    ↓
【Action】チームに共有、実際の改善に活かす
```

---

## Step 1: 自社HPを分析する（25分）

### 1.1 HPのURLをCursorに入力

以下のプロンプトをコピーして、自社HPのURLを入れ替えてください：

```
@wajima.mdc を参照しながら進めてください

まず、以下のURLの自社HPを分析してください：
https://example.com（←自社HPのURLに変更）

分析観点：
- ファーストビューの印象（最初に目に入る情報は何か）
- ナビゲーションの分かりやすさ
- CTAの明確さ（ユーザーに何をしてほしいか）
- 情報の優先順位（重要な情報が上に来ているか）
- デザインの統一感
```

### 1.2 競合と比較する（オプション）

```
競合サイトと比較して、自社HPの強み・弱みを分析してください。

競合サイト：
- https://competitor1.com（←競合のURLに変更）
- https://competitor2.com（←競合のURLに変更）

比較観点：
- 第一印象
- 情報の見つけやすさ
- 差別化ポイント
```

---

## Step 2: 改善案を出す（20分）

### 2.1 改善案を5つ提案してもらう

```
分析結果をもとに、自社HPの改善案を5つ提案してください。

それぞれについて以下を説明してください：
1. なぜ改善が必要か（現状の問題点）
2. どう改善するか（具体的なアクション）
3. 期待される効果
```

### 2.2 優先順位をつける

```
提案された5つの改善案について、
「効果」と「実装の容易さ」の観点から優先順位をつけてください。

優先度が高い順に並べ替えて、理由も説明してください。
```

---

## Step 3: 新HPのプロトタイプを作成する（50分）

### 3.1 HTMLファイルを生成

```
改善案をもとに、新しいHPのプロトタイプをHTMLで作成してください。

要件：
- モダンでシンプルなデザイン
- レスポンシブ対応（スマホでも見やすい）
- ファーストビューを重視
- CTAを明確に
- CSSはインラインまたはstyleタグで記述

ファイル名：new-hp.html
```

### 3.2 Live Previewで確認

1. 作成された `new-hp.html` を右クリック
2. 「Show Preview」を選択
3. プレビューを見ながら調整

### 3.3 調整・修正

```
プレビューを見ましたが、以下の点を修正してください：
- （気になる点を具体的に記載）
- （色、文字サイズ、配置など）
```

---

## Step 4: 仕上げ（10分）

### 4.1 最終確認チェックリスト

- [ ] ファーストビューで伝えたいことが伝わるか
- [ ] CTAが明確か
- [ ] スマホでも見やすいか
- [ ] 情報の優先順位が適切か
- [ ] デザインに統一感があるか

### 4.2 発表準備

```
今日作成した新HPについて、以下の形式で発表準備をしてください：

1. 【Before】現状のHPの課題
2. 【改善ポイント】何を改善したか
3. 【After】新HPの特徴
4. 【Action】明日からどう活かすか

3分で話せる分量でまとめてください。
```

## 困ったら

### 「分析結果がイマイチ」

```
もっと具体的に分析してください。
特に以下の点について詳しく教えてください：
- （知りたいこと）
```

### 「改善案が抽象的」

```
改善案をもっと具体的にしてください。
「〇〇のテキストを△△に変更する」
「□□のボタンを右上から中央に移動する」
のように、具体的なアクションで示してください。
```

### 「HTMLがうまく動かない」

```
このHTMLがうまく表示されません。
エラーを確認して修正してください。
```

### 「デザインがイメージと違う」

```
デザインを以下のように変更してください：
- 色味：（希望の色）
- フォント：（希望のフォント）
- 雰囲気：（シンプル/モダン/温かみ など）
```
